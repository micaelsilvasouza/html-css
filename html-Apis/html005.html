Web Worker

Ao executar scripts em uma página HTML, a página deixa de responder até que o script seja concluído.

Um web worker é um JavaScript executado em segundo plano, independentemente de outros scripts, sem afetar o desempenho da página. Você pode continuar a fazer o que quiser: clicando, selecionando coisas, etc., enquanto o web worker é executado em segundo plano.
