Armazenamento na Web

Com o armazenamento da Web, os aplicativos da Web podem armazenar dados localmente no navegador do usuário. Ao contrário dos cookies, o limite de armazenamento é muito maior (pelo menos 5MB) e as informações nunca são transferido para o servidor.
