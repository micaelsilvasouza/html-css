Word Wrap

Quando uma palavra é muito longa como essa: PNEUMOUTRAMICROSCOPICOSSILICOVULCANOCONIÓTICO, elas são normamente increbráveis em duas linha mas a proprieda word-wrap: break-word, possibilita divila é mais linhas

Word Break

A proprieda word-break especifica como as quebras de linha devem, no caso nesse paragrafo as quebras estão ocorrendo corretamente

Mas nesse daqui as quebras estão um pouco bagunçadas por causa da propriedade word-break

Writing Mode

O writing-mode especifica como as linhas ficarão dispostas, as possibilidades são

horizontal

vertical
